# Client Lead Management System (Mini CRM)

## 📌 Overview

The Client Lead Management System (Mini CRM) is a web-based application designed to help businesses manage client information, track leads, monitor sales progress, and organize customer interactions efficiently. The system provides a centralized platform for storing and updating lead details throughout the sales lifecycle.

---

## 🚀 Features

* Add, edit, and delete client records
* Lead tracking and status management
* Search and filter clients
* Dashboard with lead statistics
* Contact information management
* Follow-up and notes tracking
* Responsive and user-friendly interface
* Data persistence using database/local storage

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/client-lead-management-system.git
```

2. Navigate to the project directory

```bash
cd client-lead-management-system
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open in browser

```text
http://127.0.0.1:5000
```
